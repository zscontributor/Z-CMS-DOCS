# Z-CMS ドキュメント

[English](../README.md) | [Tiếng Việt](README.vi.md) | **日本語**

Z-CMS はマルチテナント CMS です。**1 つのコードベースと 1 回のデプロイで、
複数の独立したサイトを運用できます**。ブランド、支店、顧客ごとに、独自の
ドメイン、コンテンツ、テーマ、設定を持つことができます。Web サイトごとに
ソースコードを複製・保守する必要がないため、開発者は単一のコードベースから
すべてのサイトへリリースでき、企業は共有インフラ上で新しい支店やブランドを
数分で立ち上げられます。

このリポジトリには、Z-CMS のユーザー、拡張機能の開発者、運用担当者、
マーケットプレイスのパブリッシャー向けのドキュメントが含まれています。
ドキュメントは **[docs.z-cms.org](https://docs.z-cms.org)** で公開されています。

## 開発

Node.js 22.12 以降および pnpm 10 以降が必要です。

```bash
pnpm install
pnpm dev
```

ローカルサイトは `http://localhost:4321` で確認できます。

## 検証

```bash
pnpm check
pnpm build
```

## コンテンツ

ドキュメントは `src/content/docs/<locale>/` にあり、対象読者別に分類されています。

- `users` — サイト所有者、編集者、管理者
- `developers` — コア、プラグイン、テーマの開発者
- `operators` — セルフホスティングおよび本番環境の運用担当者
- `marketplace` — パブリッシャーおよびパッケージ配布担当者
- `contributing` — ドキュメントへのコントリビューションガイド

ベトナム語（`vi`）がデフォルトのロケールです。英語ページは `en` 配下で
同じ相対パスを使用します。

## デプロイ

本番環境の正規 URL は `https://docs.z-cms.org` です。ビルド出力は `dist/` に
生成される完全な静的サイトで、任意の静的ホスティングプロバイダーへ
デプロイできます。
