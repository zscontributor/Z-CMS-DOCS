Z-CMS Marketplace は Z-SOFT が管理・運営しています。開発者はプラグインやテーマを公開でき、ユーザーは Admin からパッケージを検索してインストールできます。

Marketplace は、拡張機能の検索、公開者の本人確認、パッケージの審査、バージョン情報、失効リストを管理します。ランタイムが信頼するのは、あらかじめ固定した公開鍵で署名を検証できたパッケージだけです。

公開者は `zcms keygen` で作成した鍵を使ってパッケージに署名します。Marketplace は受け付け時に公開者の署名を検証し、パッケージをスキャンして 2 つ目の署名を追加します。パッケージが公開されるのは、担当者の審査で承認された後です。Z-CMS はインストール前に Marketplace の署名を検証します。

## 掲載に必要な情報

- 説明とスクリーンショット。
- バージョンと変更履歴。
- Z-CMS との互換性。
- 要求する権限。
- データの取り扱いとプライバシーポリシー。
- サポート URL とドキュメント URL。
- セキュリティ連絡先。

アップロード、審査、署名、公開の手順は、[パッケージを公開する](/ja/marketplace/publishing/)を参照してください。
