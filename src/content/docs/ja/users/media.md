Media Library では、ファイルをフォルダーごとに整理できます。画像バリエーションは worker がバックグラウンドで生成するため、アップロードはその処理を待たずに完了します。

## よく使う操作

- 複数のファイルをドラッグ＆ドロップしてアップロードします。
- キャンペーンやコンテンツタイプ別にフォルダーを作成します。
- 複数のファイルを選択して移動または削除します。
- アクセシビリティと SEO のために代替テキストを設定します。

公開 Web サイトで使用するファイルには、機密情報を含めないでください。

## 閲覧、検索、フィルター

通常は、現在開いているフォルダーの内容だけが表示されます。現在位置はパンくずリストで確認できます。検索は Media Library 全体が対象で、現在のフォルダーだけには限定されません。画像または文書で絞り込むことができ、1 ページあたり最大 24 件が表示されます。

## フォルダーとメタデータ

Media の更新権限を持つユーザーは、フォルダーの作成・名前変更・移動、ファイルの名前変更・移動、代替テキストの編集ができます。ファイルやフォルダーの削除には `media:delete` 権限が必要です。削除する前に、そのファイルがコンテンツ、テーマ設定、サイトロゴで使われていないか確認してください。

## 画像バリエーション

元画像は表示されるのにサムネイルがない場合は、アップロードを繰り返さず、**Jobs** と worker の状態を確認してください。画像バリエーションの生成がまだ完了していない可能性があります。

:::tip
内容を表すファイル名、安定したフォルダー構成、文脈上の目的を伝える代替テキストを使用し、SEO キーワードの詰め込みは避けてください。
:::
