Z-CMS Admin では、コンテンツ、メディア、テーマ、プラグイン、サイトメンバーをまとめて管理できます。

:::note[技術用語の表記について]
コマンド名、API、マニフェストのフィールド名、画面上のラベルは、製品と照合できるよう原文のまま記載します。プラグイン、テーマ、ランタイムなど一般的な技術用語はカタカナで表記し、それ以外の説明は自然な日本語で記述します。
:::

## ログイン

ローカル環境では、`http://localhost:3101` を開いて Admin にアクセスします。初期サンプルデータを使用している場合は、メールアドレス `admin@z-cms.org`、パスワード `admin123` でログインできます。2 要素認証を有効にしている場合は、認証アプリに表示されるコードも入力してください。

## 最初のステップ

1. **Settings** でサイト名、タイムゾーン、言語を確認します。
2. **Content** を開き、最初のコンテンツを作成します。
3. **Media** に画像をアップロードします。
4. **Appearance** でメニューと有効なテーマを確認します。
5. **Users** でメンバーを招待し、適切なロールを割り当てます。

:::tip
**Administrator** ロールは、サイト全体を管理する必要があるユーザーにのみ割り当ててください。
:::
