zAI を使うと、自然言語の指示でページとブログを管理できます。zAI は、現在のアカウントに付与された権限を超える操作を実行できません。すべての操作は Z-CMS API を経由し、権限確認と監査ログの対象になります。

## 対応する操作

- ページとブログの一覧表示または検索。
- Draft 状態のコンテンツ作成。
- タイトル、スラッグなど対応フィールドの編集。
- コンテンツの公開と非公開への変更。
- 明示的な確認後の削除。

## 正確な指示を書く

指示には、コンテンツタイプ、状態、言語、変更する値を含めてください。例:

```text
「メンテナンスのお知らせ」という日本語の blog draft を slug「maintenance-notice」で作成してください。
```

```text
タイトルに「サービス」を含む公開中の page を一覧表示してください。
```

操作後は **Content** を開き、特にスラッグ、言語、内部リンク、SEO フィールドを確認してください。

## 削除操作

削除を依頼すると、zAI は処理を一時停止して **Confirm deletion** を表示します。実行前に対象をもう一度確認し、検索結果から対象を一意に特定できない場合はキャンセルしてください。

:::caution
会話にはシークレット、認証情報、機密性の高い個人情報を入力しないでください。zAI はコンテンツ操作を支援する機能であり、編集・承認手続きの代わりにはなりません。
:::

権限不足と表示された場合は、適切な権限のアカウントを使うか Editor/Admin に依頼してください。1回の作業のためだけに広いロールを付与しないでください。
