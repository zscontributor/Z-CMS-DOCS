## Tìm đúng tài liệu

- Nếu bạn quản lý website trong Admin, bắt đầu tại **Người dùng**.
- Nếu bạn phát triển integration, plugin hoặc theme, chọn **Developers**.
- Nếu bạn triển khai Z-CMS trên hạ tầng riêng, chọn **Vận hành**.
- Nếu bạn muốn phát hành package, chọn **Marketplace**.
